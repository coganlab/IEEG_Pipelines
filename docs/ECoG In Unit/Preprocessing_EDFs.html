Check triggers when EDF files are uploaded to Box

  1. After running tasks with a subject, we submit Task Name, Date, and Begin/End time of each task to the neurology team in the Neurology Timestamp doc.
  2. A member of the neurology team will parse the long recordings into smaller EDF files that contain the range of data for each task. See Timestamps instructions
  3. The EDF files will be uploaded to Box Sync\CoganLab\ECoG_Task_Data\TaskUploadDir
  4. Once uploaded, we need to validate that the correct range was extracted.
  5. Please take a look at view_triggers.m for further instructions for loading and visualizing the trigger channel contained in the edf file. The triggers should look like the following (Insert window of triggers from EDF section)
  6. Once EDF is validated, you can move to Box Sync\CoganLab\ECoG_Task_Data\Cogan_EDF
