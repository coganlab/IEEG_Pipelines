Preprocessing Set Up:

 

Create folder “InUnit Preprocessing” to house all preprocessing files:

 

Within “InUnit Preprocessing”:

  •      Create folders for each of the participants (D##) you will be preprocessing
  •       Download maketrigtimes and view_triggers matlab scripts from box
    • maketrigtimes: Box Sync\ECoG_Recon\matlab_code\ecog_preprocessing
    • view_triggers: Box Sync\CoganLab\ECoG_Task_Data\TaskUploadDir
    • Startup.mat will also be saved here. See page 2 for details

 

Within the participant folder, create folders for each task completed/to be preprocessed

 

ECoG preprocessing:

 

  • Setup your Matlab to include the following:

 

 

globalFsDir will need to be set specific to your own computer

This code will need to be run before beginning EACH preprocessing session

 

 

To Begin Preprocessing:

 

  • Download EDF file (location: Box Sync\CoganLab\ECoG_Task_Data\TaskUploadDir) for the participant/task you will be preprocessing
    • Place EDF file into designated participant (D##)/ task folder from page 1

 

  • Download trialdata.mat  for same participant/task (location: Box Sync\CoganLab\ECoG_Task_Data\Cogan_Task_Data)  
    • Place into same folder as EDF (TIMIT preprocessing requires all trialdata.mat files for each block and practice to be downloaded)

 

  1. In matlab: make current folder desired partcicipant (D##)>task folder
  2. Run startup.mat
  3. Open ecog_preprocessing.m

 

Each section of ecog_preprocessing.m is meant to be run one at a time, you can highlight the section and press F9 to run. 

 

*Add a new case for each task/recording sections:

 

  • Copy and paste the most recent case (example figure A) and replace purple text with current participant information and file locations.
  • Once filled in, highlight lines ‘cd’ to ‘rec’ and press F9 to run.
