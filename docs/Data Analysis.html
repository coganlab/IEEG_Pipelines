System Setup:

       Running both Python MNE and FreeSurfer on Windows requires a WSL (Windows Subsystem Linux):

 

  1.  FreeSurfer runs on all OS's besides Windows, to accomodate for this we need to use a WSL
    1. A WSL is a Linux virtual machine that runs simultaneously on a Windows machine with shared file systems
    2. We want to do this so that python MNE can reference the files containing our FreeSurfer reconstructions - using the Environment Variables (paths) created by the Freesurfer Environment
  2. Install WSL

    2. Changing your settings
    3. Changing your Securities
    4. obtaining a distribution 
  3. Installing X-Server

  2. Testing your GUI
    1. FSL - MRI viewer 
  3. Install FreeSurfer on the WSL
    1. Traverse to the Freesurfer website and select a Linux distribution
    2. Copy the link to the distribution
    3. wget the link 
  4. Install miniconda or anaconda on the WSL 
    1. After installation make sure to say "yes" when prompted on whether you want conda or anaconda to initialize in your bash window 
  5. Pip install the MNE environment on the WSL
  6. Acitvate the MNE environment
    1. anaconda or conda activate mne 
  7. Make sure the Subject Directories are pointing to the correct locations
    1. export SUBJECTS_DIRS & export FREESURFER_HOME - will change these pointers
