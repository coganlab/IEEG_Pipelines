Uniqueness Point

 

Lab: Cogan Lab

 

Laptop: Lenovo PC (Zeus or Hades)

 

Program: Psychtoolbox

 

Location: C:/Psychtoolbox_Scripts/uniqueness_point

 

Filename: uniqueness_point_no_delay_button

 

Equipment Used: Photodiode, microphone, 2X amplifiers, speakers

 

Command to run: uniqueness_point_no_delay_button(subject, practice, startblock)

     subject = 'D#'

     practice = 0 (no) or 1 (yes) 12 trials

     startblock = 1 for beginning, or n if interrupted and resuming task 

 

Duration: 4 blocks total (120 trials per block); ~40 minutes total

 

Instructions:

If you see the cue Yes/No  please press the left arrow key for a word OR the right arrow key for a non-word. 

 

If you see the cue Repeat, please repeat the word/nonword.

 

Respond as quickly was possible. 

 

This task contains 4 blocks and each block lasts about 10 minutes. You can stop at any time, and we encourage you to take breaks in between each block.
